# 如何删除 Gboard 表情历史记录
只是一个笔记，因为我谷歌了一大顿，没什么有用的结果。<br>
<br>
**仅适用于具有Root权限的设备。**<br>
反正在我的索尼Xperia 5 iv XQ-CQ44上是好用的。<br>
删除这个文件：<br>
`/data/data/com.google.android.inputmethod.latin/databases/expression-history.db`
* 防止Gboard记住最近使用的表情：<br>
删除上述文件后，创建一个同名的文件夹即可。<br>
